可在相应的工具栏中找到可用命令。每个命令都会显示属性页面以进行编辑和插入。操作的结果将作为特征插入到特征树中。

所有特征都支持参数化功能，一旦模型或依赖特征或选择发生更改，它们将自动更新。

特征可以像SOLIDWORKS中的任何其他标准特征一样进行编辑、抑制、重新排序和删除。

其中一些特征暴露出可以修改的尺寸，可以与设计表关联或通过API进行控制。
