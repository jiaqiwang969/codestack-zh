该功能将实体转换为曲面。

![将实体转换为曲面的属性管理器页面](convert-solid-to-surface-page.png)

* 选择要转换的实体或实体组
* 点击绿色勾号

新功能将添加到特征管理器树中。

![特征管理器树中的实体转曲面功能](solid-to-surface-feature.png)

所有选择的实体将被相应的曲面替换。该功能是完全参数化的。如果一些形成实体的基础特征被修改，曲面特征也会被修改。
